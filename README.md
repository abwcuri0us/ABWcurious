# 🌌 ABW Curious Learning Platform

![ABW Logo](/public/logo.png)

Welcome to the absolute evolution of the **ABW Curious Ecosystem**. This repository houses a globally persistent, high-performance web platform built explicitly on the Next.js 14 App Router Architecture utilizing hyper-modern Glassmorphism UI components across a purely mathematical 3D-space rendered environment. 

## 🚀 True Scale Architecture & Features
The system is partitioned into two massive operating zones:

### 1. The Cosmos App Suite (Client Ecosystem)
Every module inside the `/apps` directory is built utilizing strict internal Javascript mechanics mapping pure physics and geometry parameters natively rendering deep computation.

- **Scientific Cosmo Calculator:** Features complex `Regex` overrides injecting true calculus (`sin`, `cos`, `tan`, `log`, `π`, Exponentials, and Square Roots) dynamically inside the UI.
- **Timeline Engine (Calendar Array):** Computes deep algorithmic sequences calculating exact days per month, determining absolute 35-block UI grids, and highlighting live `Date()` arrays autonomously.
- **Mistral AI Web-Speech Hologram:** The system connects purely via physical Microphone (`webkitSpeechRecognition`) routing user voice text across the Mistral API and returning via robotic synthetic voice feedback `speechSynthesis`. The 3D CSS Face maps literal cursor-tracking matrices.
- **Space Simulation Environment:** The universal Background runs localized geometry utilizing absolute CSS loops tracking dozens of physical elements (Sun, Satellites, Comets, Nebulas) dynamically.

### 2. The Prisma SQL Admin Network (`/dashboard`)
Designed via literal Next.js 14 Server Actions (`app/actions.ts`), the Master Admin Grid utilizes a purely structured SQLite Database rendering God-Mode access to global operations perfectly bypassing Vercel Cache lags.

- **Persistent Registration & Auth Engine:** Prevents unregistered access strings, authenticating strict Prisma `.findUnique` payloads internally ensuring encrypted access.
- **Global Chat Socket:** Polls absolute database fetches creating pseudo-websockets mapping multi-user telemetry into a centralized, persistent communication stream natively formatted.
- **Client Architecture Drops:** Clients instantly deploy Forms dropping CRM/ERP requirements or Personnel recruitment arrays straight into the SQL server for massive administrative observation.
- **God-Mode Surveillance:** Admins physically manipulate User Names, Contact Vectors, and execute physical Wiping of records via God-node execution.

## 🛠️ Execution & Deployment Vectors

**1. Clone the Central Repository**
```bash
git clone <repository_url>
cd ABWcurious
```

**2. Inject Node Process Modules**
```bash
npm install
```

**3. Mount the Prisma Local Architecture**
```bash
npx prisma generate
npx prisma db push
```

**4. Spin up the Local Host Server**
Execute the nextjs local array natively:
```bash
npm run dev
```
Navigate to `http://localhost:3000` to physically interact with the ecosystem.

---

### Deployment to Vercel/Production
This application is completely cleared for production launch natively via the standard Vercel pipeline. 

No special localized Node.js servers are required. SQLite mappings are automatically synthesized by Prisma during the compilation build (`npm run build`). Just push your exact files to GitHub, bind it to your Vercel Node project, and deploy the entire structure simultaneously.
